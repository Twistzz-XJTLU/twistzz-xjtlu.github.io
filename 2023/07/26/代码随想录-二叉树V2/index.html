144. 二叉树的前序遍历 迭代法

思路

  • 利用栈 因为是前序遍历 所以中间节点要先处理
  • 首先加入最上面的中间节点 然后弹出栈顶元素 把栈顶元素加入到结果集
  • 接下来就要处理中间节点的左右节点
  • 前序遍历是中左右 但是在入栈的时候要先加入右节点 为什么
  • 因为先入栈的元素后出来 我们先加入了右节点的元素 后加入了左节点的元素
  • 到时候出栈添加元素的时候就可以先添加左元素了
  • 每一次获取 node 后就要 判断右 和左是否有节点 如果有继续入栈 相当于判断该节点的底下一层

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def preorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        if not root:
            return []
        stack=[root]
        res=[]
        while stack:
            node=stack.pop()
            res.append(node.val)
            if node.right:
                stack.append(node.right)
            if node.left:
                stack.append(node.left)
        return res

94. 二叉树的中序遍历 迭代法

思路

  • 中序遍历和前序遍历不一样 前序遍历的顺序是中左右，先访问的元素是中间节点，要处理的元素也是中间节点，
  • 那么再看看中序遍历，中序遍历是左中右，但我们实际先访问的是二叉树顶部的节点，然后一层一层向下访问，直到到达树左面的最底部，再开始处理节点（也就是在把节点的数值放进 result 数组中），这就造成了处理顺序和访问顺序是不一致的。
  • 用 cur 指针表示当前当前访问的节点
  • 进入循环的条件 如果栈不为空或者当前元素不是空就需要进入循环
  • 如果当前元素不是空那就代表我们还可以继续访问前元素的左孩子(不一定有值但是需要访问 因为你也不知道有没有值) 当前元素就可以入栈 然后访问左孩子
  • 如果当前元素是空那就代表此节点是空 此时栈顶元素是当前元素的父节点 弹出栈顶元素 把此元素放到结果数组中
  • 然后我们看当前元素的有孩子 指针指向当前元素的右孩子

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        if not root:
            return []
        res=[]
        cur=root
        stack=[]
        while stack or cur:
            if cur:
                stack.append(cur)
                cur=cur.left
            else:
                cur=stack.pop()
                res.append(cur.val)
                cur=cur.right
        return res

145. 二叉树的后序遍历 迭代法

思路

再来看后序遍历，先序遍历是中左右，后续遍历是左右中，那么我们只需要调整一下先序遍历的代码顺序，就变成中右左的遍历顺序，然后在反转 result 数组，输出的结果顺序就是左右中了

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def postorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        if not root:
            return []
        res=[]
        stack=[root]
        while stack:
            node=stack.pop()
            res.append(node.val)
            if node.left:
                stack.append(node.left)
            if node.right:
                stack.append(node.right)
        return res[::-1]

102. 二叉树的层序遍历

思路

  • 利用队列 先入先出
  • 首先判断是否有 root 没有就直接返回 有就加入队列
  • level 用来存储每层变量 没遍历完一层 就想 res 中添加一次
  • 那么如何判断是否遍历完一层了呢
  • 首先我们已经加入了 root 元素 因此进入循环 最外层的是 queue 循环 如果没有了 那就代表我们已经遍历完所有元素了
  • 内层的 for 循环就代表着每一次的元素 为什么呢
  • 因为我们在后面判断了当前元素的左右孩子是否有 如果有就添加到 queue 中
  • 所以当我们遍历完当前一层后 此时队列中的元素就是下一层中所有的元素 所以 for 循环的长度是 len(queue)
  • 也就是我们每遍历完一层就会创建一个新的 level 记录要遍历的下一层的元素

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []
        queue,res=[root],[]
        while queue:
            level=[]
            for i in range(len(queue)):
                node=queue.pop(0)
                level.append(node.val)
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)
            res.append(level)
        return res

226. 翻转二叉树

思路

  • 递归三部曲 递归函数参数和返回值 结束条件 单层递归逻辑 同时既然是递归那么一定会在递归函数中在此调用自己
  • 本次采用后序遍历 先把根节点的左右孩子进行翻转 最后交换根节点的左右孩子
  • 对左孩子进行翻转 然后右孩子 因为是层层递归 运行完前这两行函数 那么就一定代表着左右孩子都已经完成了翻转
  • 所以交换左右孩子就可以了

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def invertTree(self, root: Optional[TreeNode]) -> Optional[TreeNode]:
        if not root:
            return
        self.invertTree(root.left)
        self.invertTree(root.right)
        root.left,root.right=root.right,root.left
        return root

101. 对称二叉树

思路

  • 本体定义函数稍有不同 入参不再是一个 root 变成了 left 和 right 这样便于判断
  • 进入函数 如果左节点没有 右节点有(注意是节点不是孩子) 或者左有 右没有 不满足条件
  • 如果左右节点值不相等肯定也不满足条件
  • 如果左右节点都是空满足条件
  • 注意左右节点相等并不意味着就要返回 Ture 为什么呢 因为当前左右节点相等并不意味着他的子节点也是相等的
  • 所以还要继续往下判断 那么如何判断呢
  • 并不是简单的看当前孩子的左右还是是否相等 因为现在已经有了两个节点 那么意味着下面就是四个节点
  • 所以我们要这样判断: 左节点的左孩子和右节点的右孩子 左节点的右孩子和右节点的左孩子
  • 判断上面这里的是否都是 True 最后返回结果

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def isSymmetric(self, root: Optional[TreeNode]) -> bool:
        if not root:
            return True
        return self.traversal(root.left,root.right)
    def traversal(self,left,right):
        if not left and right:
            return False
        elif not right and left:
            return False
        elif not left and not right:
            return True
        elif left.val!=right.val:
            return  False
        left_res=self.traversal(left.left,right.right)
        right_res=self.traversal(left.right,right.left)
        return left_res and right_res

104. 二叉树的最大深度

思路

  • 使用后序遍历 左右中
  • 如果当前节点是空那么就返回零
  • 如果不是空然后判断当递归获取当前节点左孩子的深度和当前节点右孩子的深度
  • 最后去两者大的然后加 1
  • 为什么要加 1 呢? 因为这个 1 代表当前节点的高度
  • 举个例子 如果当前节点没有左右孩子 那么我们就要返回 1 给上一层函数
  • 二叉树节点的深度：指从根节点到该节点的最长简单路径边的条数或者节点数（取决于深度从 0 开始还是从 1 开始）
  • 二叉树节点的高度：指从该节点到叶子节点的最长简单路径边的条数或者节点数（取决于高度从 0 开始还是从 1 开始）
  • 而根节点的高度就是二叉树的最大深度，所以本题中我们通过后序求的根节点高度来求的二叉树最大深度。

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        return self.traversal(root)
    def traversal(self,node):
        if not node:
            return 0
        left_depth=self.traversal(node.left)
        right_depth=self.traversal(node.right)
        return 1+max(left_depth,right_depth)

111. 二叉树的最小深度

思路

  • 思路和最大深度类似 但是并不完全一样
  • 首先还是后续遍历左右中
  • 如果当前节点是空就返回 0
  • 然后递归求出左右孩子的最小深度
  • 但是 并不能直接 return 1+min(left_depth,right_depth)
  • 为什么? 因为我们要注意最小深度的定义 是根节点到最近的叶子结点的距离 叶子结点是左右节点都为空
  • 举个例子 一个根节点 然后左边什么也没有 右边有好几层的孩子
  • 此时如果我们直接返回 那么得到的最小深度就是 0+1 但是此时的最小深度并不是 1
  • 因此在我们求出来左右孩子的深度后要进行判断
  • 如果左孩子是空右孩子不是空 那么我们就要返回右孩子的深度 因为我们不能算左边 左边不是叶子节点 因为最小深度一定从在右孩子中出现
  • 反过来也一样
  • 那么如果都不为空呢? 这就代表该节点左右都有孩子 此时我们就要取左右孩子中最小的
  • 那么如果都是空呢? 这就代表该节点就是叶子节点我们也走1+min(left_depth,right_depth) 这句代码 因为左右孩子都是 0 1+0 也就是返回 1 当前叶子节点的高度

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def minDepth(self, root: Optional[TreeNode]) -> int:
        return self.traversal(root)
    def traversal(self,node):
        if not node:
            return 0
        left_depth=self.traversal(node.left)
        right_depth=self.traversal(node.right)
        if node.left is None and node.right is not None:
            return 1+right_depth
        if node.left is not None and node.right is None:
            return 1+left_depth
        return 1+min(left_depth,right_depth)

222. 完全二叉树的节点个数

思路

  • 本体类似于求二叉树的最大深度
  • 采用后序遍历 左右中
  • 如果当前节点是空那么就返回 0
  • 不是空的话就求当前节点的左节点的个数和当前节点的右节点的个数
  • 最后左节点的个数+右节点的个数+1 是最终结果
  • 为什么要加 1 呢 因为这个 1 就代表当前节点 所以要加上自己

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def countNodes(self, root: Optional[TreeNode]) -> int:
        return self.traversal(root)
    def traversal(self,node):
        if not node:
            return 0
        left_num=self.traversal(node.left)
        right_num=self.traversal(node.right)
        return left_num+right_num+1

110. 平衡二叉树

思路

  • 本体使用后序遍历 因为要判断高度 所以左右中 使用后序遍历
  • 递归函数终止条件 首先如果该节点是空节点就返回 0 因为高度是 0
  • 然后计算左节点和右节点的高度 如果高度是-1 就代表左节点已经不是平衡二叉树了 返回他的高度没有意义
  • 因此左右节点如果是-1 的话继续向上层返回-1 (一层一层向上) 最后肯定是 False 因为我们要判断是否是平衡二叉树
  • 然后进入单层递归逻辑 排除以左右节点为根节点的树不是平衡二叉树的情况后 我们就可以判断以当前节点作为根节点的树是否是平衡二叉树
  • 如果左右节点高度大于 1 那么就返回-1 否则就返回1+max(left_height,right_height)
  • 为什么要返回刚才那个式子呢 因为我们还要向上返回 1 代表当前节点的高度 为什么要选最大值 我们要判断平衡二叉树 所以要去最大的高度

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def isBalanced(self, root: Optional[TreeNode]) -> bool:
        if self.traversal(root)==-1:
            return False
        return True
    def traversal(self,node):
        if not node:
            return 0
        left_height=self.traversal(node.left)
        if left_height==-1:
            return -1
        right_height=self.traversal(node.right)
        if right_height==-1:
            return -1
        if abs(left_height-right_height)>1:
            return -1
        else:
            return 1+max(left_height,right_height)

257. 二叉树的所有路径

思路

  • 本体采用前序遍历 中左右 首先我们定义一个 res 和 path 分别存放最终结果和每条路径
  • 本体递归函数的终止稍有不同 不再是判断当前节点是否是空 本体我们判断当前节点的左右节点是不是都是空
  • 刚进入函数的时候我们就要把当前节点计入 path 中 那万一当前节点是空呢? 不会有这种情况 因为我们下面已经判断了 如果不是空才会进入下一轮递归
  • 如果都是空那么代表我们到了叶子阶段 是收集结果的时候了 因此如果左右都是空 此时我们手机结果把 path 加入到 res 中
  • 然后我们分别判断左右孩子是否是空 如果不是空我们就分别继续递归左右孩子 向下寻找 直到找到叶子节点
  • 注意 本体中含有回溯 一个是显示回溯一个是隐式回溯
  • 如果我们用显示回溯 调用 traversal 函数的 path 就传递本身 这意味着什么呢意味着当前节点的左孩子也加入到了 path 中 当我们遍历完左孩子的时候我们就要把左孩子 pop 出来
  • 为什么要 pop 因为遍历完左孩子 我们就要遍历右孩子 但是左孩子此时还在 path 中 path 中左孩子的前一个是根节点 所以我们 pop 后前一个节点是根节点 我们要以根节点为基准继续遍历右孩子
  • 隐式调用什么意思呢 就是我们调用 traversal 函数的 path 不传递 path 本身 我们传 path[:] 这代表什么
  • 这意味着左节点不会进入到我们的 path 中 只会进入到 path[:] 他和我们的 path 无关 因此我们也就不用再 pop 了 path 最后一个元素就是根节点 我们在遍历右孩子的时候直接遍历就行了

代码 1 显示回溯

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def binaryTreePaths(self, root: Optional[TreeNode]) -> List[str]:
        if not root:
            return []
        res,path=[],[]
        self.traversal(root,res,path)
        return res
    def traversal(self,node,res,path):
        path.append(node.val)
        if node.left is None and node.right is None:
            spath="->".join(map(str,path[:]))
            res.append(spath)
            return
        if node.left is not None:
            self.traversal(node.left,res,path)
            path.pop()
        if node.right is not None:
            self.traversal(node.right,res,path)
            path.pop()

404. 左叶子之和

思路

  • 本体要求左叶子之和 有些特殊 平时都是判断本节点 但是根据当前节点我们不能判断是否是叶子节点
  • 所以我们要通过父节点来进行判断
  • 采用后续遍历 先递归调用左右孩子 分别求出左孩子的左叶子之和 和右孩子的左叶子之和
  • 然后我求出当前节点的左叶子之和
  • 如果当前节点的左孩子有 而且左孩子的左孩子是空 左孩子的右孩子是空 那就代表当前节点的左孩子是我们要求的点
  • 用mid_sum记录当前值
  • 最后返回三者之和

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def sumOfLeftLeaves(self, root: Optional[TreeNode]) -> int:
        if not root:
            return 0
        left_sum=self.sumOfLeftLeaves(root.left)
        right_sum=self.sumOfLeftLeaves(root.right)
        mid_sum=0
        if root.left is not None and root.left.left is None and root.left.right is None:
            mid_sum=root.left.val
        return left_sum+right_sum+mid_sum

513. 找树左下角的值

思路

  • 本体使用层序遍历比较简单
  • 记录每一层的第一个值
  • 进行层序遍历
  • 最后遍历的一层的第一个值就是我们要的答案

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def findBottomLeftValue(self, root: Optional[TreeNode]) -> int:
        if not root:
            return 0
        queue=[root]
        res=0
        while queue:
            for i in range(len(queue)):
                node=queue.pop(0)
                if i==0:
                    res=node.val
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)
        return res

112. 路径总和

思路

  • 本体采用精简代码版本比较好理解和书写
  • 采用前序遍历
  • 通过当前节点是空那么就返回False 为什么? 因为通过遍历到了当前节点 意味着targetSum还没有被减到0
  • 那就意味着我们还需要找到一个值让targetSum去减 但是当前节点都是空了 肯定targetSum 不能再减少了 所以就返回False
  • 然后接着判断 如果当前节点是叶子几点也就是当前节点的左右孩子都是空
  • 而且当前节点的值就是targetSum的值 那么就代表着targetSum减去当前节点的值是0 所以找到了这条路径 于是就返回True
  • 然后接着判断左右孩子 如果左右孩子中有一个是True 就返回True

代码(精简版本)

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def hasPathSum(self, root: Optional[TreeNode], targetSum: int) -> bool:
        if not root:
            return False
        if root.left is None and root.right is None and root.val==targetSum:
            return True
        return self.hasPathSum(root.left,targetSum-root.val) or self.hasPathSum(root.right,targetSum-root.val)

113. 路径总和 II

思路

  • 本题使用前序遍历 判断逻辑是看所剩的值是否和当前节点的值一样大并且该节点是叶子节点
  • 首先要定义path和res变量 分别存储每条路径和最后结果
  • 进入循环 如果当前节点是空的话那个就什么都不敢 res不用动
  • 所以经过是否为空判断后 那么此节点一定不是空 所以加入path路径中 并且total_sum减少相应的值
  • 如果该节点是叶子节点并且total_sum是0了(因为我们total_sum先减去了当前节点的值)
  • 然后判断左右节点
  • 最后一定要记得path pop 为什么要pop?
  • 因为我们当前节点一开始被加入到了path中 path会一直存储这这个值
  • 假设我们当前遍历的节点是左孩子 遍历完之后 我们就要遍历右孩子了
  • 但是在遍历右孩子之前 我们要把左孩子pop出去 因为这不是一条路径 我们遍历右孩子的到时候path最后的值应该是根节点 而不是左节点

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> List[List[int]]:
        path,res=[],[]
        self.backtracking(root,targetSum,path,res)
        return res
    def backtracking(self,node,total_sum,path,res):
        if not node:
            return 
        path.append(node.val)
        total_sum-=node.val
        if node.left is None and node.right is None and total_sum==0:
            res.append(path[:])
        self.backtracking(node.left,total_sum,path,res)
        self.backtracking(node.right,total_sum,path,res)
        path.pop()

106. 从中序与后序遍历序列构造二叉树

思路

  • 本题采用前序遍历 首先我们确定中间节点 然后递归调用函数 确定中间节点的左右节点
  • 我们知道了中序和后续 这意味着什么
  • 意味着根据后续的最后一个元素可以判断当前谁是根元素
  • 我们根据当前根元素 获取当前根元素代中序数组中的索引 这样我们就知道中序数组怎么切了
  • 为什么要切 因为切了之后我们才知道数组的哪一部分是当前节点的左孩子们 当前节点的右孩子们
  • 切完中序数组之后我们就要切后续数组 那么后序数组怎么切呢
  • 中序数组 因为我们知道谁是根节点了 我们就可以找到根节点的中序数组中的位置 然后切割 那后序数组呢
  • 后序数组我们要根据中序数组切割后的左孩子们构成的数组的长度进行切割 因为我们切割前序数组后会用变量存储左前序数组和右前序数组
  • 后序数组切割的时候 左后序数组是从零开始到左前序数组的长度 为什么?
  • 因为左前序数组的长度我们已经知道了 就意味着当前节点的左孩子的数量我们知道了
  • 牢记等式: 中序数字和数量=后序数组的数量 所以后序左数组就是[0:len(left_inorder)]
  • 那后序右数组呢 len(left_inorder):-1 一定要注意是到-1 而不是最后
  • 因为最后一个元素是当前的根节点 不是我们需要的
  • 为什么要先切中序数组再切后序数组

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def buildTree(self, inorder: List[int], postorder: List[int]) -> Optional[TreeNode]:
        if not postorder:
            return None
        root_val=postorder[-1]
        root=TreeNode(val=root_val)

        split_index=inorder.index(root_val)
        left_inorder=inorder[0:split_index]
        right_inorder=inorder[split_index+1:]
        
        left_postorder=postorder[0:len(left_inorder)]
        right_postorder=postorder[len(left_inorder):-1]

        root.left=self.buildTree(left_inorder,left_postorder)
        root.right=self.buildTree(right_inorder,right_postorder)
        return root

105. 从前序与中序遍历序列构造二叉树

思路

  • 和上一题思路已经
  • 主要前序切得时候一定要加上1 因为1是最开始的 要跳过 所以都要加1

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def buildTree(self, preorder: List[int], inorder: List[int]) -> Optional[TreeNode]:
        if not preorder:
            return None
        root_val=preorder[0]
        root=TreeNode(val=root_val)

        split_index=inorder.index(root_val)

        inorder_right=inorder[split_index+1:]

        preorder_left=preorder[1:1+len(inorder_left)]
        preorder_right=preorder[1+len(inorder_left):]

        root.left=self.buildTree(preorder_left,inorder_left)
        root.right=self.buildTree(preorder_right,inorder_right)
        return root

654. 最大二叉树

思路

  • 首先求出最大值和最大值的下表
  • 然后根据最大值简历根节点 根据最大值下表切割nums
  • 然后递归调用函数求出当前节点的左孩子和当前节点的右孩子
  • 最后返回根节点

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def constructMaximumBinaryTree(self, nums: List[int]) -> Optional[TreeNode]:
        if not nums:
            return None
        max_val=max(nums)
        max_index=nums.index(max_val)
        root=TreeNode(val=max_val)
        root.left=self.constructMaximumBinaryTree(nums[:max_index])
        root.right=self.constructMaximumBinaryTree(nums[max_index+1:])
        return root

617. 合并二叉树

思路

  • 本题采用前中后序遍历都可以 那我们就采用前序遍历 中左右 比较好理解
  • 递归终止条件 如果第一颗二叉树的当前节点是空那么我们就返回第二颗二叉树的当前节点
  • 第二颗二叉树的当前节点也有可能是空 但是没关系是空我们也返回了
  • 同样的 第二棵树当前节点是空我们就返回第一树的当前节点 为什么要真么做?
  • 因为我们要计算的是两棵树的当前节点的和 如果其中有一颗是空那么我们就不用计算了直接返回那个不是空的就可以了
  • 然后计算两节点的和 递归调用当前节点的左右子树 最后返回根节点
  • 注意本题我们没有新建一个根节点 而是直接在左树上面进行了修改 减少了内存占用

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def mergeTrees(self, root1: Optional[TreeNode], root2: Optional[TreeNode]) -> Optional[TreeNode]:
        if not root1:
            return root2
        if not root2:
            return root1
        root1.val+=root2.val
        root1.left=self.mergeTrees(root1.left,root2.left)
        root1.right=self.mergeTrees(root1.right,root2.right)
        return root1

700. 二叉搜索树中的搜索

思路

  • 本题比较简单直接利用二叉搜索树的性质
    • 采用前序遍历
  • 递归函数的终止条件 如果当前节点是空 那么就返回None
  • 如果当前节点的值等于要求的值 那么久返回当前节点 (这点和上一点可以合并)
  • 如果判断当前节点的值和需要求的值得大小关系 然后递归调用
  • 注意我们要用变量存储这最后结果 因为我们要返回这个结果

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def searchBST(self, root: Optional[TreeNode], val: int) -> Optional[TreeNode]:
        if not root:
            return 
        if root.val==val:
            return root
        if root.val>val:
            res=self.searchBST(root.left,val)
        else:
            res=self.searchBST(root.right,val)
        return res

98. 验证二叉搜索树

思路

方法1

  • 设定一个最大值 不断更新最大值进行判断
  • 首先递归函数的终止条件 如果当前节点是空 那么空节点也可以是二叉搜索树 所以返回True
  • 然后使用中序遍历 先判断做左孩子们的情况 然后判断中间节点 然后判断右孩子们的情况
  • 如果当前节点大于最大值 要把当前节点的值设为最大值
  • 如果当前节点小于等于最大值 证明这不是二叉搜索树
  • 因为我们先遍历了左孩们 而孩子们的值得最大值应该小于当前节点(也就是根节点的值) 等于也不行
  • 如果前节点小于等于最大值 我们就返回False
  • 然后我们判断右孩子们 判断右孩子和判断中间节点一样
  • 因为右孩子也是一定需要大于目前为止的最大值 如果不大于就说明不是二叉搜索树
  • 最后判断是否左右孩子都满足 要使用and

方法2

  • 使用中序遍历 转换成数组判断
  • 但是使用这种方法一定不要忘记判断相等情况
  • 因为如果遍历后的数组中存在连续的两个数相等的情况那也不是二叉搜索时
  • 因此需要if res[i]<=res[i-1] 这样

代码

设定最大值，进行比较

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def __init__(self):
        self.max_val=float("-inf")
    def isValidBST(self, root: Optional[TreeNode]) -> bool:
        if root is None:
            return True
        left_condition=self.isValidBST(root.left)
        if root.val>self.max_val:
            self.max_val=root.val
        else:
            return False
        right_condition=self.isValidBST(root.right)
        return left_condition and right_condition

530. 二叉搜索树的最小绝对差

思路

方法1

  • 通过一个格外的pre指针
  • 注意 这个指针和result一定要定义为全局变量 因为这两个是不可变类型 不想list一样
  • 如果当前是空那么就返回 什么也不干
  • 然后递归调用左孩子们
  • 递归调用完左孩子们后 如果pre不是空的 那么我们就要去当前节点与pre节点的差 和result中小的值
  • 然后pre节点等于当前节点 相当于移动pre节点
  • 然后递归调用右孩子们
  • 最后返回result结果

方法2

  • 使用中序遍历
  • 变成list 然后在list中进行中判断

代码

使用额外指针

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def __init__(self):
        self.result = float('inf')
        self.pre = None

    def traversal(self, cur):
        if cur is None:
            return
        self.traversal(cur.left)  # 左
        if self.pre is not None:  # 中
            self.result = min(self.result, cur.val - self.pre.val)
        self.pre = cur  # 记录前一个
        self.traversal(cur.right)  # 右

    def getMinimumDifference(self, root):
        self.traversal(root)
        return self.result

利用中序递增，结合数组

# Definition for a binary tree node.

#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def getMinimumDifference(self, root: Optional[TreeNode]) -> int:
        res=[]
        self.traversal(root,res)
        result=float("inf")
        for i in range(1,len(res)):
            result=min(result,res[i]-res[i-1])
